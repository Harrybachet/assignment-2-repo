How to Apply CSS to HTML

To include an external CSS file in an HTML file, use the <link> element within the <head> section. For example: 

<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, use the <script> element with the src attribute. For example: 

<script src="script.js"></script>
To write JavaScript code directly within an HTML file, place the code within the <script> tags. For example: 
<script>
    console.log('Hello, world!');
</script>
